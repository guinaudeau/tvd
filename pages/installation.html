Installation

Requirements

tvd relies on the following set of command line tools. It was successfully tested with the following versions:

  • lsdvd
  • MPlayer
    • MPlayer 1.1-4.2.1 (C) 2000-2012 MPlayer Team
  • MEncoder
    • MEncoder 1.1-4.2.1 (C) 2000-2012 MPlayer Team
  • sndfile_resample
    • libsamplerate-0.1.8 (c) 2002-2008 Erik de Castro Lopo
    • libsndfile-1.0.25
  • vobsub2srt
    • tesseract 3.02.02
    • leptonica-1.69
  • tessdata

Installation

Once those tools are installed, installing tvd is as easy as 

pip install tvd
